# PDF 无损去水印工具

一个基于 Python 和 PyMuPDF 的“外科手术级” PDF 去水印脚本，专为处理包含复杂图文排版、水印与正文严重重叠的 PDF 文件而设计（已默认适配某五字大学下载下来的期末试卷）。

与传统的“区域注销（Redaction）”或“图片覆盖”方法不同，本工具直接深入 PDF 底层**内容流 (Content Stream)**，从字节层面精准抹除水印的文本绘制指令。**100% 保护水印下方重叠的正常文字、高清图表和矢量布局**，是处理“文字叠文字”复杂场景的终极解决方案。

## ✨ 核心特性

* **外科手术级精准清除**：直接修改底层字节流（替换字符串指令为 `() Tj`），绝对不误伤、不吞噬任何被水印遮挡的底图和排版。
* **智能正则适配**：默认支持正则表达式，自动识别并清除包含任意数字序列的动态水印（如 `XJTLU Academic Use Only by [学号]`）。
* **全图层净化**：不仅修改底层文字流，还会同步扫描并剥离悬浮在页面上方的独立文本框和注释层（Annotations）水印。
* **自动化批量处理**：一键扫描当前目录及一级子目录下的所有 `.pdf` 文件。
* **安全且极致优化**：自动生成 `（去水印版）` 后缀的新文件（不覆盖原文件），并在保存时执行深度元数据清理与极致压缩，保持文件小巧。

## 🛠️ 环境依赖

运行此脚本需要 Python 3.x 环境，并安装 `PyMuPDF` 库：
```bash
pip install PyMuPDF
```

## 🚀 快速开始

1. 将需要去水印的 PDF 文件放入脚本所在的同一文件夹（或其一级子文件夹）内。
2. 在终端或命令行中运行脚本：
```bash
python remove_watermark.py
```

3. 脚本会自动扫描并处理文件。处理完成后，你将在原文件旁边看到形如 `文件名（去水印版）.pdf` 的高清无损去水印文件。
4. 或者下载我打包好的exe运行，目录要求同脚本一致。

## 🧠 技术原理 (Why it works better?)

传统使用 PyMuPDF 的 `add_redact_annot()` 方法本质上是在 PDF 上“用橡皮擦擦除一个矩形区域”，这会导致该坐标框内的**所有图层**（包括底层正常文字、矢量线条）全部被破坏。

**本工具的解决方案：**
1. 调用 `page.clean_contents()` 整合页面零碎的绘图流。
2. 提取 PDF 底层机器代码指令（如 `(XJTLU Academic Use Only by 1234567) Tj`）。
3. 使用二进制正则匹配（Regex on Bytes），将目标文本的绘制内容替换为空字符串 `()`。
4. 写回 PDF，从而在不改变任何几何排版的前提下，让水印“隐形”。

## ⚙️ 自定义配置

如果你需要去除其他类型的水印，只需在代码中修改以下两个变量：

* **`static_keywords`**: 填入固定不变的水印短语列表。
* **`dynamic_pattern_byte`**: 编写字节级别的正则表达式，用于捕获包含动态变量（如时间、学号、ID）的水印。
